# Verificação de Idade / Age Check

Este é meu primeiro programa em Python.

O projeto realiza uma verificação simples da idade do usuário, informando se ele é maior ou menor de idade com base no valor inserido.

## 🛠️ Tecnologias utilizadas
- Python

## 📌 Funcionalidades
- Solicita o nome do usuário
- Solicita a idade
- Informa se o usuário é maior ou menor de idade

## ▶️ Como executar
1. Execute o arquivo `Age_check.py`
2. Insira seu nome
3. Insira sua idade

## 📚 Objetivo
Projeto desenvolvido para praticar conceitos básicos de:
- Entrada de dados (`input`)
- Condicionais (`if/else`)
- Saída de dados (`print`)
